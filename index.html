This application uses AJAX technic to fetch data from:

  • jsonplaceholder
  • codecamp.org
  • TMDB
  • International Space Stationcurrent location.
  • NASA OPEN DATA PORTALglobal temperature.

◉ You can fetch data from each api by clicking any of the buttons on your left.

◉ Data fetched include posts, tasks/todos, users from jasonplaceholder API, Inpsirational quotes from Quotes free API, Internation space Station current position and much more.

◉ I've used leaflet API for rendering the map. Checkout the list of APIs I'm working with on github page .

◉ Tabular data displayed in this project comes from National Aeronautics and Space Administration NASA particularly from the Goddard Institute of Space Studies.This CSV file include the combined global average of land surface, air and sea surface water temperature from 1880 to-date. chat.js has been used for graph representation.

◉ Global-mean monthly, seasonal, and annual means, 1880-present, updated through most recent month: TXT, CSV Northern Hemisphere-mean monthly, seasonal, and annual means, 1880-present, updated through most recent month: TXT, CSV Southern Hemisphere-mean monthly, seasonal, and annual means, 1880-present, updated through most recent month: TXT, CSV Zonal annual means, 1880-present, updated through most recent complete year: TXT, CSV.

◉ Feel free to explore.
