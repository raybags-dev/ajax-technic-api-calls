This application uses AJAX technic to fetch data from:

  • jsonplaceholder
  • codecamp.org
  • TMDB
  • International Space Stationcurrent location.

You can fetch data from each api by clicking any of the buttons on your left

Data fetched include posts, tasks/todos, users from jasonplaceholder API, Inpsirational quotes from Quotes free API, Internation space Station current position and much more. I've used leaflet API for rendering map, Checkout the list a APIs I'm working with on github page

Feel free to explore.

More APIs like "weather APIs", "foods", and much more will be intergrated here as well. Don't forget to check-in.
